## 📖 项目简介

Koch 是一款专业的摩尔斯电码学习训练工具，采用经典的 **Koch 方法**（Koch Method）帮助用户高效掌握摩尔斯电码。与传统学习方法不同，Koch 方法从一开始就以标准速度（20 WPM）播放字符，通过逐步增加字符数量来降低学习难度，使学习者能够直接建立"声音-字符"的条件反射。

### 什么是 Koch 方法？

Koch 方法由德国心理学家 Ludwig Koch 提出，是目前最科学、最高效的摩尔斯电码学习方法：

- **高速起步**：从第一课开始就使用标准速度（通常 18-25 WPM）
- **渐进学习**：初期只学习 2 个字符，掌握后逐步添加新字符
- **条件反射**：避免"计数点划"，建立直接的听觉识别能力
- **科学有效**：大幅缩短学习时间，通常 2-3 个月即可达到实用水平

## ✨ 主要特性

- 🎯 **标准 Koch 训练流程**：严格遵循 Koch 方法的科学训练步骤
- 🎨 **简洁友好界面**：直观的操作体验，专注于学习本身
- 📝 **多种字符训练模式**：支持 Koch-LCWO 字符集，以及英文、数字、标点字符集
- 🔊 **可调节音频参数**：自定义音调频率、速度（WPM）、Farnsworth 间隔等
- 📊 **可视化数据分析**：每日练习活跃度显示，多维度追踪训练准确率变化趋势
- 📈 **波形图实时显示**：支持音频播放时实时显示方波波形图
- 💾 **本地数据存储**：学习记录本地保存，保护隐私

## 📚 使用指南

### 第一步：开始训练

- 启动应用后，点击"播放"按钮
- 第一课将练习字符 **K** 和 **M**
- 听到摩尔斯电码后，输入对应字符
- 当正确率达到 90% 以上时，解锁下一课

### 第二步：逐步进阶

- 每课增加 1-2 个新字符
- 建议每天练习 20-30 分钟
- 不要急于求成，确保每课都达标再进入下一课

### 第三步：调整参数

- 字符速度（Character Speed）：建议 20 WPM
- 有效速度（Effective Speed）：可从 5 WPM 开始
- 音调（Tone）：500-800 Hz 之间选择舒适的频率
